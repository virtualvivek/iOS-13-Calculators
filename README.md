## Basic Calculator
An ios 13 style `Light theme` Calculator .

Written in html5 and css3 for web browsers .

Easily Embbed into pages and use solved results.

### Basic Calculator
To use values returned by the calculator-
```js
use {{c_value}}

Eg <input type="text" value="{{c_value}}"/>
```
OR through ng tag-
```js
ng-model="c_value"
```
